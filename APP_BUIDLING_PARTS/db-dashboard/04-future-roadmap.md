# 04-future-roadmap.md

## Future Roadmap & Next Steps

### Coming Features
- Embedded AI agent panel (AutoSQL, query audit, explain)
- Workflow chaining and programmable dashboards
- Panel marketplace/registry for user-contributed widgets
- Opencode/MCP-powered graph/metrics dashboards
- Cloud and file-based DB import/export (offline safe)
- Advanced security: workspace scopes, airgapped audit, fine-grained sharing

### Expansion Ideas
- Multi-modal media (file explorer, text-edit, browser panel, etc)
- Panel debugger/trace/recorder for automating panel UIs
- SSO/openid panel context for workspace-aware workflows
- NixOS-native DB mount panel

### Implementation Queue
- Panel SDK & starter templates
- Panel/plugin hot-reloading
- Tablet/mobile UX optimizations
- User config auto-save/restore

### Feedback and Community
- All enhancements must be documented in this folder via markdown before code
- User feedback queue at each feature.
