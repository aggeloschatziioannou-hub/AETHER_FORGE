Key design decisions:

  • Next.js routes proxy — Canvas panels talk to /api/proxy/*, not directly to Python. Avoids CORS, keeps auth in one place.
  • FastAPI runs as sidecar — Started alongside npm run dev via a start.sh. Docker Compose ready later.
  • SQLite direct — Python's built-in sqlite3 for SQLite databases. No need for MCP for simple SQLite access.
  • MCP for remote DBs — When you add PostgreSQL/MySQL support, proxy connects via MCP Python SDK.
  • Lazy connections — Proxy only opens DB connections when a panel sends a request. Idle panels = zero connections.
