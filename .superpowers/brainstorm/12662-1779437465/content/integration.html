Two directions for the DB dashboard

A

A page in the existing Next.js app

New route like /db in your aether-forge-app. Sits alongside the canvas. Uses the same Tailwind theme, same layout, shares auth/state. One app to maintain.

B

Standalone tool

Separate app/SwiftUI/Tauri. Opens a port, talks to MCP servers. Independent from the canvas project. Can be a desktop app if desired.
